# Lorem Ipsum Generator

Esse modulo gera texto Lorem Ipsum com Java. Ele utiliza uma implementação bem simples que gera lipsums com contagem de palavras específico, contagem de parágrafos ou de palavras por parágrafo.  
This module generates Lorem Ipsum Text with Java. It utilizes a very simples implementation that generates lipsums with specific word count, paragraph count or words p/paragraph count.  

## How to use it?

1.) Export it to your workspace or folder  
2.) In your module-info.java write:  

```Java
module example {
	//...
	requires com.generator;
}
```

3.) If in Eclipse, fix the build path of your project  
4.) Use it calling the LoremIpsumGenerator in your main method
